SeyalRun / Settings / Command ACL

Command ACL

Filters run top-down per session; the first match wins. Blocked commands are killed on the remote line and recorded.

protect-logsdeny  rm -rf /var/log* · applies to all hosts deny
no-package-changesdeny  (apt|yum|dnf) (install|remove) · role: user, guest deny
allow-service-restartsallow  systemctl (restart|reload) * · group: ops allow
legacy-freezedeny  * on legacy-app · disabled while migration runs off

Default action for unmatched commands: allow & record · audited

Type

System stack only (SF on Apple hardware) — zero font bytes, instant first paint, native rendering on every OS. Monospace is reserved for machine values: IPs, hashes, terminal.

Engineering notes — why it’s fast

  • No new framework. Same Vue 3 + Pinia + hash router; this is a skin + component pass. Backend untouched.
  • Zero font downloads. System stack; the biggest render-blocking asset class is simply gone.
  • Route-level code splitting. Terminal (xterm) and playback bundles load only when opened — dashboard TTI drops.
  • Virtualized tables for hosts/sessions/audit past ~100 rows; hover states via CSS only.
  • Materials in CSS. Translucency is backdrop-filter; shadows are two layers max; no images.
  • Motion budget. 120–180 ms ease-out on interaction only; honors prefers-reduced-motion.
  • ⌘K command palette — jump to any host, run a playbook, open a session; the keyboard is the fastest UI.
  • Skeletons, not spinners — perceived speed on the dashboard’s 15-query payload.
